## 2.大小写和内容
#### 2.1 大小写

* **SDK** 和 **API** 全部大写，
* 其他软件、语言、接口和库的名称按照官方网站名称确定大小写，如 Altizure、 JavaScript、HTML、GraphQL、OpenGL 等。

#### 2.2 内容

* **marker** 和 **Sandbox** 不用翻译

* 汉字、英文和数字混杂时应空一格，如：**这里的 camera 参数，是指 sandbox 初始化的时候的相机位置。**
* 请注意拼写的正确性
* 在插入图片时注意包括图片库和文档中的后缀名一律用小写，如 .png

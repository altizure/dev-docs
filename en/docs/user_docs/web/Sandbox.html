    Sandbox(divname, sandboxOptions)

    Display, manipulate, interact with multiple altizure projects, and multiple imported gl elements.

    new Sandbox(divname, sandboxOptions)

    Creates an instance of Sandbox.

    Parameters:
    Name Type Description
    divname string

    div element's name which contains the canvas
    sandboxOptions object

    options for initialization

    Properties
    Name Type Attributes Description
    altizureApi object

    api settings

    Properties
    Name Type Attributes Description
    key string

    developer key
    apiUrl string <optional>

    developer key, default https://api.altizure.com/graphql
    region string <optional>

    'CN' or 'Inter'. default 'CN'
    api object <optional>

    engine api config (encrypted)
    camera object <optional>

    open animation settings

    Properties
    Name Type Attributes Description
    poseTo pose <optional>

    animation start camera pose (lat, lng, alt, north, tilt)
    flyTo pose <optional>

    animation end camera pose
    renderItems object <optional>

    what to render

    Properties
    Name Type Attributes Default Description
    earth bool <optional>

    render earth
    earthUseTexture bool <optional>
    		 true

    render satellite images
    featureInView bool <optional>
    		 true

    render altizure featured projects
    orbitRing bool <optional>
    		 false

    render earth orbit ring
    background.transparent bool <optional>
    		 false

    gl background transparent
    background.opacity bool <optional>

    gl background transparent

    gl background transparent
    background.image bool <optional>

    gl background transparent

    pick(event) → {LngLatAlt}

    Pick the point on earth at a specified mouse event position.

    Parameters:
    Name Type Description
    event MouseEvent

    mouse/touch event
    Returns:
    LngLatAlt -
    • contact point with earth
    Inherited From:
    • Earth#pick
    Example

    pick earth point by mouse 

    let earth = new altizure.Earth('page-content', options)
let domElement = earth.domElement
domElement.addEventListener('mousedown', function (event) {
  if (event.button === 0) { // left button
    let pt = earth.pick(event)
    console.log('earth.pick', pt)
  }
}, false)

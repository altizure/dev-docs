    TagMarker(markerOptions)

    Tag marker always facing towards the camera and maitains its screen space size. Note: the pivot is at the center of the image.

    new TagMarker(markerOptions)

    Parameters:
    Name Type Description
    markerOptions object
    Properties
    Name Type Description
    imgUrl string

    marker icon url
    Example

    construction 

      let tagMarker = new altizure.TagMarker({
    imgUrl: '../public/assets/img/meta_description.png',
    // icon position
    position: {lng: 113.9390549031058, lat: 22.53601411862488, alt: 59.03098709697936},
    // scene
    sandbox: sandbox,
    scale: 1 // icon size
  })

    Do depth test or not. If false, always in front.

    Overrides:
    • SpriteMarker#depthTest
    Example

    Change depth test status 

     spriteMarker.depthTest = true // do depth test
 spriteMarker.depthTest = !spriteMarker.depthTest // flip the status

    Set if the marker is interactable. Set to true to respond to mouse events.

    Inherited From:

 marker.interactable = !marker.interactable // flip interactable status

    The length of the pin underneath the sprite.

    Overrides:

    scale

    The scale.

    texture

    Texture path image url.

    Overrides:
    • SpriteMarker#texture
    Example

    Change imgUrl

    textureAsync

    Set texture with handlers. {data: imageUrl, onSuccess: handler, onFail: handler}

    Example

    Change imgUrl with callback handlers 

     tagMarker.textureAsync = {
   data: '../public/assets/img/meta_description.png',
   onSuccess: function () { console.log('the imgUrl has been changed successfully!') },
   onFail: function () { console.log('Fail to change the imgUrl') }
 }

    on(eventType, handler)

    Register an event with a handler. Note: an event can only has one handler.

    Parameters:
    Name Type Description
    eventType string

    'click', 'mouseover', 'mouseenter', 'mouseleave'
    handler function

    event handler
    Inherited From:
    • Marker#on
    Example

    Register interaction events 

     marker.on('click', function (event) {
   console.log('click on marker ', marker, event)
 })
 marker.on('mouseover', function (event) {
   console.log('mouse over marker ', marker, event)
 })
 marker.on('mouseenter', function (event) {
   console.log('mouse enter marker ', marker, event)
 })
 marker.on('mouseleave', function (event) {
   console.log('mouse leave marker ', marker, event)
 })
 marker.interactable = true // the marker must be interactable for events to take effects
